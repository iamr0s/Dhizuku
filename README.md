# Dhizuku

## 介绍

参考Shizuku的设计思想，分享 DeviceOwner (设备所有者) 权限给其余应用

## 支持版本

Android 5.0 ~ 13

## 致开发者（接入Dhizuku API）

- [Dhizuku API](https://github.com/iamr0s/Dhizuku-API.git)
